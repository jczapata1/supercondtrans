<h1 align="center">TDGL</h1>

<p align="justify">
Time-Dependent Ginzburg-Landau simulations (TDGL) of thin superconducting films using <a href="https://py-tdgl.readthedocs.io/en/latest/">py-tdgl</a>. Supports static and dynamic vortex configurations under uniform, plateau, and stripe-domain magnetic field profiles, with applied transport current and current-voltage sweep mode.
</p>

## Workflow

| Notebook | Description |
|---|---|
| `Device.ipynb`     | Build the TDGL mesh. |
| `Fields.ipynb`     | Compute magnetic field and vector potential profiles (Uniform, Plateau, Domains). |
| `Simulation.ipynb` | Run the TDGL solver (Static, Dynamic, Sweep). |
| `Animation.ipynb`  | Generate animations from simulation snapshots. |
